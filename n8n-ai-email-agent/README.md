# AI Email Operations Agent (n8n)

Chat-based workflow for automating email operations
(sending emails, inbox retrieval) using an AI Agent, OpenAI, and Gmail tools.

## Features
- Natural language → action pipeline (chat input → tool execution)
- Multi-tool orchestration within n8n (Gmail read & send)
- Dedicated error workflow with alerting for failed executions

## Notes
- Workflow screenshots are publicly available in this repository.
- JSON export is provided on request only for security reasons.
